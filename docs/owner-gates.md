# Owner Gates

Ask for explicit Owner approval before:

- publishing or pushing this private subproduct to any remote
- changing repository visibility
- printing, copying, or storing secret values
- changing global Codex / Claude Code / OpenCode configuration
- editing shell profiles
- publishing packages or creating marketplace/Homebrew/npm/cargo releases
- uploading project data, compact checkpoints, logs, or usage history
- changing license or public legal wording

Allowed by default:

- local profile file edits
- local tests and scans
- user-level AiPlus profile files under `~/.config/aiplus/`
- metadata-only secret-broker status/list/doctor checks
